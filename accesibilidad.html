Compromiso con la accesibilidad

Este sitio web se ha desarrollado siguiendo los principios de accesibilidad universal, con el objetivo de garantizar que cualquier persona, independientemente de sus capacidades o dispositivo, pueda acceder a los contenidos de forma cómoda y eficiente.

La estructura y el diseño siguen las Pautas de Accesibilidad al Contenido Web (WCAG 2.1) establecidas por el W3C (World Wide Web Consortium).

Medidas aplicadas en el desarrollo

1. Estructura semántica

Todas las páginas utilizan etiquetas HTML5 adecuadas (header, nav, main, section, article, footer), favoreciendo la lectura por parte de lectores de pantalla y mejorando la jerarquía del contenido.

2. Contenido alternativo

Todas las imágenes cuentan con descripciones mediante el atributo alt, permitiendo que los usuarios con discapacidad visual comprendan su contenido.

3. Formularios accesibles

Los formularios incluyen etiquetas label asociadas correctamente a cada campo, facilitando la navegación mediante teclado y su identificación por tecnologías asistivas.

4. Contraste y legibilidad

Se ha aplicado una paleta de colores con suficiente contraste entre texto y fondo, siguiendo las recomendaciones del nivel AA de las WCAG 2.1, para mejorar la legibilidad.

5. Diseño responsive

El diseño se adapta automáticamente a distintos tamaños de pantalla mediante técnicas Flexbox y @media queries, garantizando una correcta visualización en dispositivos móviles, tabletas y ordenadores.

Navegación mediante teclado

El sitio puede explorarse completamente sin necesidad de ratón. A continuación se muestran las principales combinaciones de teclas:

Combinación Función
Tab / Shift + Tab Desplazarse entre enlaces y campos del formulario.
Enter Activar el enlace o botón seleccionado.
Alt + Inicio Ir directamente a la página de inicio (en navegadores compatibles).

Contacto y sugerencias

Si detectas algún problema de accesibilidad o deseas proponer una mejora, puedes comunicarlo a través del formulario de contacto.
